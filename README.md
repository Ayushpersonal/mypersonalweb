# Ayush Kumar - Premium Portfolio

A lightweight, high-performance portfolio website built with React, Vite, and Framer Motion.

## 🚀 Key Features

- **Performance First**: Optimized for low-end systems with conditional 3D loading and "Fast Mode" toggle.
- **Premium Aesthetics**: Dark-themed glassmorphism design with a high-end feel.
- **Fail-Safe Architecture**: Uses React Error Boundaries to prevent crashes if external 3D assets (Spline) fail to load.

## ✨ Premium Animations (Lightweight 3D)

- **Magnetic Buttons**: Interactive buttons that smoothly follow your cursor, providing a tactile, tactile experience.
- **Parallax Tilt cards**: Project and skill cards that tilt in 3D based on mouse position, using performant transformation logic.
- **Smooth Scroll**: Seamless navigation between sections using optimized frame-rate animations.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS v4, Framer Motion
- **Icons**: Lucide React
- **3D**: Spline (Lazy-loaded with fallback)
- **Deployment**: Optimized for Vercel/Netlify

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
